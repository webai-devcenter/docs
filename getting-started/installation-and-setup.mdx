## What to Expect When Available

When webAI becomes publicly available, here's what the installation process will look like:

### Download webAI

<Steps>
  <Step title={<h2>Download the webAI Installer</h2>}>
    - Visit the webAI download page.
    - Download the webAI Installer (acts as your package manager for all webAI tools).
    - Add the webAI Installer to your Applications folder.

    **What You'll Get:**
    - A single installer providing access to both Navigator and Companion.
    - A centralized tool for managing webAI components.
  </Step>
  <Step title={<h2>Launch the Installer and Install Applications</h2>}>
    - Open the webAI Installer from your Applications folder.
    - Click "Install Both" to install Navigator and Companion together.
    - Installation typically takes 3-5 minutes.
    - Both applications are installed for an integrated AI development experience.

    **System Integration:**
    - Navigator: Your AI workflow development environment.
    - Companion: Your personal AI chat interface.
  </Step>
  <Step title={<h2>Initial Setup</h2>}>
    - Launch Navigator post-installation.
    - Complete the account creation form.
    - Dependencies will install automatically upon first launch.
    - Respond to any macOS system prompts (security permissions, network access).

    **Account Creation:**
    - Provide your email address (login credential).
    - Set a secure password.
    - Optionally, add your organization name.

    **System Configuration:**
    - Navigator configures itself for your specific device.
    - Default settings applied automatically.
    - No manual configuration required for basic setup.
  </Step>
  <Step title={<h2>Verify</h2>}>
    **What Success Looks Like:**
    - Navigator dashboard displays with Featured Templates visible.
    - Companion launches and connects to Navigator automatically.
    - Both applications are running and ready for AI development.
  </Step>
</Steps>

## System Preparation

### Ensure Your Device is Ready

**Check Compatibility:**

- Review our System Requirements to verify your Mac can run webAI optimally
- Ensure adequate storage space for installation and AI models
- Verify stable internet connection for download and setup

**Prepare Your Environment:**

- Close unnecessary applications to free up system resources
- Ensure you have administrator privileges for installation
- Consider temporarily disabling antivirus during installation

### Network Considerations

**Potential Issues to Be Aware Of:**

- **VPN Interference:** Active VPN clients can interfere with local network discovery
- **Corporate Firewalls:** May block required network connections
- **Network Permissions:** Applications will need network access for full functionality
